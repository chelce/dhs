The Locker Code
Easy
Jamie forgot their locker combination but remembers it follows a simple pattern. Can you figure it out and help them open their locker?

What's the combination?
Locker Code Clues
1. The combination has exactly 3 numbers.
2. Jamie's name is "JAMIE" - count the letters.
3. The first number equals the number of letters in Jamie's name.
4. The second number is one more than the first number.
5. The third number is the same as the first number.

The Color Pattern
Easy
Your art teacher left a coded message using colored squares that repeat in a pattern. If you can figure out the pattern, you can predict what color comes next!

What color is in position 23?
Color Pattern Clues
1. The pattern is: Red-Blue-Red-Blue-Yellow
2. This pattern repeats every 5 colors.
3. Position 1 = Red, Position 2 = Blue, Position 3 = Red...
4. Position 5 = Yellow, then it starts over.
5. To find position 23, figure out where it falls in the pattern.

The Detective Challenge
Hard
Someone ate the last slice of pizza at lunch, and there are 4 suspects. Each person makes a statement, but only one person is telling the truth. Use deductive reasoning to solve the mystery.

Who ate the pizza?
Detective Evidence
Alex says: "Blake ate the pizza."
Blake says: "I was in math class when it happened."
Casey says: "Drew definitely ate it."
Drew says: "Casey is lying about me."
Key Rule: Only ONE person is telling the truth.
Logic Check: If Drew tells the truth, then Casey lies, so Drew didn't eat it. But who did?

Hard

What time works for everyone?
Schedule Constraints

Carlos: Work shift ends at 2:00pm (free after)
